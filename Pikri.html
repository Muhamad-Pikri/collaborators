Belajar JavaScript

Panduan singkat belajar JavaScript, dari dasar hingga konsep lanjutan.

Pengantar JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web lebih dinamis dan interaktif.

Keunggulan JavaScript

  • Dapat dijalankan di browser tanpa instalasi tambahan.
  • Digunakan untuk manipulasi halaman web, validasi, dan banyak lagi.

Cara Penulisan JavaScript

Script dapat ditulis dalam tag <script>:

<script>
    alert("Hello, World!");
</script>

Variabel JavaScript

Variabel adalah tempat penyimpanan data. Gunakan kata kunci var, let, atau const: 

let name = "John";
const age = 30;

Gunakan let untuk variabel yang nilainya dapat berubah dan const untuk nilai tetap.

Fungsi JavaScript

Fungsi adalah blok kode yang dapat dipanggil kapan saja:

function greet(name) {
return "Hello, " + name;
}
console.log(greet("Alice"));

Objek JavaScript

Objek digunakan untuk menyimpan berbagai properti dan metode:

const person = {
firstName: "John",
lastName: "Doe",
age: 30,
fullName: function() {
  return this.firstName + " " + this.lastName;
}
};

String dan Metode String

String adalah teks yang diapit tanda kutip. Beberapa metode string:

let greeting = "Hello, JavaScript!";
console.log(greeting.toLowerCase()); // hello, javascript!
console.log(greeting.indexOf("JavaScript")); // 7
console.log(greeting.slice(7, 17)); // JavaScript

Array dan Metodenya

Array digunakan untuk menyimpan banyak nilai dalam satu variabel. Contoh:

const fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits[0]); // Apple

Beberapa metode array:

fruits.push("Orange"); // Tambah elemen
fruits.pop(); // Hapus elemen terakhir

Math dan Random

Beberapa metode Math untuk operasi matematika:

Math.random(); // Random number
Math.pow(2, 3); // 8

Set dan Metodenya

Set adalah koleksi nilai unik:

let numbers = new Set([1, 2, 3, 3]);
console.log(numbers); // {1, 2, 3}

Beberapa metode Set:

  • add(): Menambahkan elemen ke dalam Set.
  • delete(): Menghapus elemen dari Set.
  • has(): Mengecek keberadaan elemen.
  • clear(): Menghapus semua elemen.
